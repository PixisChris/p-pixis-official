員工人數逾兩萬人，為全球背光模組主要的生產廠商，主要客戶有友達、群創、夏普、三星、LG 等等。

人工派發 IP 後使用 Excel 紀錄 MAC 及其他內網設備資訊與使用狀況，易有疏 漏之外也不夠及時，導致管理上麻煩且不好掌握使用狀況與資訊。

因應公司的資安要求，IT 人員需要定時盤查設備及設備安裝的防毒軟體版本， 但設備量龐大且單以防毒伺服器上的資訊不足以即時反應未安裝防毒的設備， 需要人工交叉比對並產出報表。再來，針對沒安裝防毒的設備沒有相對應的防 護措施，僅政策宣導，效果有限。

透過 PIXIS ATHENAC 系統基本的 IPAM 功能可以使 IT 人員輕易掌握內網所有 IP 的使用狀況與 MAC 的相關資訊，且自動記錄 Log 讓管理者即時查詢，掌握內網狀況。

使用 ARP 機制自動封鎖未授權的設備，提高內網設備可視度和內網設備的管控程度。

透過「外部資料」整合與「Pre-Check」功能來達到自動盤查內網設備資訊 的功能，立即顯示該設備是否未安裝的防毒軟體等管理者自訂規則，如不符合規則自動封鎖，大幅優化管理者效率。

整合 802.1x 與 IPAM 功能，且提供 Radius 服務，讓已授權的 MAC 可以透過 802.1x 進入內網， 達到安全的 802.1x 管制以及內網可視度和內網安全。
